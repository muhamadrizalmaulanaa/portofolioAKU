👨‍💻 Tentang Saya

Halo! Nama saya Rizal Maulana, seorang software developer yang berdomisili di Indonesia. Saya memiliki minat besar pada pengembangan aplikasi berbasis web dan sistem backend.

Saya terbiasa menggunakan Laravel, Python, CodeIgniter, JavaScript, serta mengelola database dengan MySQL dan SQLite. Saya juga suka bereksperimen dengan teknologi baru dan open source.

Saat ini saya sedang mengembangkan beberapa proyek pribadi dan aktif mempelajari praktik terbaik dalam pengembangan perangkat lunak.

Saya percaya bahwa teknologi bisa menyelesaikan banyak masalah, dan saya ingin jadi bagian dari solusinya.
